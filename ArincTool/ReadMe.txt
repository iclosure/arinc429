
Arinc429版本说明：

=========== V2.3.0.release ============
  1. 全面修复所有BUG；
  2. 经过严格测试规范验证，达到稳定版本。

=========== V2.2.1 ============
  1. 优化驱动和API程序模块；
  2. 修改ar_putword函数，变更为异步传输（即直到数据发送成功或超时（超时时间为100ms）才返回），用户无需自己实现数据重传代码；
  3. ArincTool增加发送数据自定义功能(*alpha version)。
  
=========== V2.2.0 ============
  1. 优化驱动和API程序模块；
  2. 修改、优化ARAPI模块函数执行速度；
  3. ArincTool增加运行控制中的“显示”选择，用户可在功能测试面板中选择进行性能测试或实时显示；
  4. 修复安装文件，支持Windows纯系统安装和运行；
  5. 将文档和各平台驱动安装文件等客户端件集成到安装文件中，用户安装软件后可在安装路径中找到所有需要的产品相关的信息和资源；
  6. 注意：由于对驱动和API等框架作了调整，考虑到兼容性问题，我们更换了驱动的GUID识别码，因此需要按要求进行相关软件的安装使用；
     对您造成的不便，表示抱歉。
     
=========== V2.1.0 ============
  1. 优化驱动程序模块；
  2. 修改、优化ARAPI模块函数执行速度；
  3. API部分增加设备类型识别，支持PCI429和USB429两种板卡接口；
  4. ArincTool的板卡选择部分可实现PCI型和USB型板卡的识别；
  5. ArincTool增加性能测试模块，针对Arinc429板卡进行满负荷性能测试。
  
=========== V2.0.0 ============
  1. 优化驱动程序模块；
  2. 修改、优化ARAPI模块函数执行速度；
  3. 优化ARAPI数据存储速度（采用自定义双端回环缓冲机制，极大提高了存储和读取的速度）；
  4. 优化ArincTool数据显示速度；
  5. 增加ARAPI接收的数据缓冲区上限值为8192个双字节（双字节，即协议字）；
  6. 更改函数ar_getnextt的返回值timetag（时间戳）的数据涵义。更正为时间戳（之前的为相邻数据时间差）；
     单位为μs（微秒）（这里的时间戳表示为系统开机到每次驱动中中断接收数据时的时间差值）；
  7. 增加多通道数据传输显示控制功能；
  8. 更改API函数接口参数类型。
